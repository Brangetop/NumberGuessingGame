# guessing-game-go

hey, this is just a small project i built while learning Golang. it's a simple game where the computer picks a random number and you have to guess it.

i'm currently teaching myself Go, so the code might be a bit messy. I wanted to practice basic stuff like loops, user input, and generating random numbers in Go.

### how to run:
if you have Go installed, just clone it and run:
`go run main.go`
